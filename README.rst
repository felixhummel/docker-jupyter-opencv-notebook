jupyter-opencv-notebook
=======================
This is the Jupyter `scipy-notebook`_ extended with OpenCV.

.. _scipy-notebook: https://hub.docker.com/r/jupyter/scipy-notebook/


Requirements

(https://docs.docker.com/compose/compose-file/#/version-21).
